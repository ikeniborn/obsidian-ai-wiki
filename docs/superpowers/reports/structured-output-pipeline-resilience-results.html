Итог

Пайплайн структурированных LLM-вызовов приведён к двум режимам: json-zod для малых объектов и framed-zod для больших Markdown/text payloads. Ошибка из лога bootstrap теперь диагностируется как пустой/non-JSON output или несовместимость response_format, с downgrade и Zod-validation после восстановления.

Глобальный jsonMode удалён из operation options: JSON response format теперь включает только json-zod runner, а framed/raw Markdown calls не наследуют JSON mode. Legacy hidden wrapWithJsonFallback удалён, чтобы fallback diagnostics оставались в одном месте.

Replay после destructive prelude заблокирован: если init --force уже выполнил WipeDomain, operation-level idle retry не повторит весь init и не сотрёт домен повторно.
