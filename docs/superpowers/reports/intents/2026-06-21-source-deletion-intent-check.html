1. Резюме требований

Objective

Удаление источника оставляет осиротевшие wiki-страницы (артефакты), портящие retrieval/граф; нет first-class операции «удалить источник»; ручная чистка трудоёмка; multi-source страницы хранят stale-вклад удалённого источника; в sidebar нет отдельной кнопки удаления.

Решение: добавить операцию Delete source — удаляет source-файл и все связанные wiki-артефакты без остатков; страницы с >1 источником не удаляются, а пересобираются последовательно на оставшихся.

Desired Outcomes

#Наблюдаемый результат
1Preview с цифрами до удаления. Модаль показывает N страниц к удалению (список) и M к пересборке (список); запуск только по явному подтверждению.
2Полная чистка артефактов. Sole-source страницы + сам source-файл удалены; убраны из source_paths/analyzed_sources, _index.md, графа, embeddings, backlinks. Ноль осиротевших.
3Multi-source страницы пересобраны на оставшихся; вклад удалённого источника исчез, wiki_sources обновлён.
4Отдельные кнопки sidebar. Format и Delete — две кнопки; обе активны только на source-файле (non-wiki, в source_paths).

Health Metrics

Граф / indexНет битых [[ссылок]], осиротевших строк _index.md, висячих рёбер графа, stale-чанков embeddings; lint чистый.
Операцииingest / lint / format / query / init продолжают работать; новый код не ломает общие фазы и RunEvent-поток.
Guardpath-traversal guard цел (validateArticlePath); без выхода за домен/vault, без удаления чужих файлов.
ПлатформаРаботает на mobile (ленивые node-builtins, requestUrl) и на обоих backend (Native / Claude).

Constraints

steeringFailure = continue + collect errors (максимум прогресса за прогон, список сбоев в конце).

(a) Карта намерения — Objective → Outcomes → Metrics

  • Корректность multi-source страниц

Desired Outcomes

  • Полная чистка (ноль осиротевших)

(d) Stop Rules — условия завершения процесса

HaltPreview нельзя посчитать точно (нечитаемый wiki_sources) или путь не прошёл validateArticlePath — не удалять наугад.
EscalateПересборка трогает файлы вне домена либо source-to-page маппинг неоднозначен (источник назван неразрешимым именем).
Done whenНа реальном vault клик Delete показывает корректные счётчики/списки; по подтверждению source-файл перманентно удалён и убран из домена, каждая sole-source страница удалена, каждая multi-source пересобрана на оставшихся; lint/check показывает ноль осиротевших / битых ссылок / stale-артефактов; сбои пересборки — в финальном отчёте, а не молча.
