OK Eval-only controlled boilerplate demotion реализован, проверен на HLD corpus и зафиксирован в repo docs + iwiki.

Краткий итог

Цель была проверить BM25 глубже и добавить controlled demotion для boilerplate/template pages без изменения runtime Query. Реализация осталась eval-only: добавлены helper-функции, demoted variants, guard reasons, report sections и live HLD comparison.

Live HLD eval: 61 markdown files, 61 page records, 442 chunk records. Лучший вариант — `weighted-lexical-demoted` с factor `0.15`: `Recall@5 = 0.76`, `nDCG@5 = 0.92`, `MRR = 1.00`, average `Overlap@5 = 0.68`. Цель `avg Overlap@5 >= 0.65` выполнена.

Raw BM25/RRF не обогнали weighted lexical. Controlled demotion factor `0.15` дал небольшой nDCG gain без Recall/MRR regression; stronger factors начали валить legacy-overlap guards.
