Требования по секциям

СекцияТребование
UX — ButtonsУбрать scope-<select>. Добавить Ask Domain (запрос к выбранному домену; disabled когда домен не выбран) и Ask Wiki (всегда активна; ConfirmModal → cross-domain *). setRunning отключает обе.
UX — Buttons › Removed scope machineryУдалить ai-wiki-scope-row, scopeToggle, desiredScope, syncScope; submitQuery(domainArg) принимает аргумент.
UX — Search Stats BlockСобытие query_stats до answerFromContext → блок над ответом. Поля: crossDomain, pagesScanned, pagesSelected, доменные поля для Ask Wiki. inputTokens дозаполняется из llm_call_stats.
UX — Search Stats › MetricsAsk Domain: домен / проанализировано (весь домен files.length) / выбрано для LLM / токены. Ask Wiki: изучено N из M / из каких / Σ проанализировано / в ответе / токены.
UX — Comment Boxtextarea width:100%, rows 2→4. Save comment справа → после клика Saved + disabled. Правка текста → снова активна. Status-span удалён.
i18nУбрать scopeAll/scopeDomain/scopeHint; ask→askDomain; добавить askWiki, confirm-, stats-лейблы, commentSavedBtn (en/ru/es).
Local ConfigУдалить lastQueryScope из LocalConfig.
Error HandlingГейт токен-обновления (только если stats-блок есть); очистка stale-блока в setRunning; defensive notice в submitQuery.
Testingeval cross-domain + single-domain assert query_stats; npm run lint; ручная UI-проверка.
