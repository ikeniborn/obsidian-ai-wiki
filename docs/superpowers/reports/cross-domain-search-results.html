§ Error HandlingПустой домен → skip; все пусты → error; изоляция per-domain embedding-fallback; abort между доменами; защитный guard на дубль-стемы.

Success Criteria

#Критерий
SC-1При sidebar=(all) + «All» запрос извлекает страницы из каждого домена, а не только domains[0].
SC-2Две стадии: stage-1 top-seedTopK на домен + BFS; stage-2 top-seedTopK среди объединённого пула; новых страниц в stage-2 не добавляется.
SC-3Ровно один вызов LLM на финальном слитом наборе.
SC-4Работает в Jaccard-режиме (без embeddings) — кросс-доменный re-rank по token-overlap.
SC-5Новых настроек нет; seedTopK управляет обеими стадиями.
SC-6Поведение одно-доменного runQuery неизменно после рефактора (snapshot-эквивалентность).
