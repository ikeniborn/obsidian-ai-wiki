# Chat Feedback (Animated Timer)

Фича улучшает UX чата: добавляет анимированный таймер в чат-пузырь во время ожидания ответа, JSONL-логирование чата, сворачиваемую секцию.

## Основные характеристики

- `chatTickHandle: ReturnType<typeof setInterval>` — интервальный таймер обновляет счётчик секунд в пузыре
- Таймер запускается при отправке сообщения, останавливается при получении ответа или ошибки
- `dispatchChat()` добавляет JSONL-запись в лог при каждом чат-туре
- Секция чата получает кликабельный заголовок-переключатель (сворачивание/разворачивание)
- При сворачивании секция скрывается, последний вопрос/ответ остаётся видим в заголовке

## Формат анимации таймера

Пузырь показывает: `"⏳ 3s"` → `"⏳ 4s"` → ... до получения ответа.
