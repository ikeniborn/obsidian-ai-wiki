# Wiki Schema — obsidian-llm-wiki

## Язык

- Основной язык: русский
- Технические термины TypeScript/Obsidian оставлять на английском (LlmClient, AsyncGenerator, RunEvent, etc.)
- Имена файлов и wiki-страниц: kebab-case, кириллица разрешена

## Структура страницы

```yaml
---
wiki_status: stub | developing | mature
wiki_sources:
  - docs/путь/к/источнику.md
wiki_updated: YYYY-MM-DD
wiki_domain: документация
tags: []
---
```

## Разделы

- H1 — название сущности
- Вводный абзац без подзаголовка
- ## Описание / ## Назначение
- ## Сигнатура / ## API (для компонентов)
- ## Ключевые решения (для паттернов)
- ## Связанные страницы (WikiLinks)

## WikiLinks

Использовать `[[имя-страницы]]` для перекрёстных ссылок между wiki-страницами.

## Запреты

- Не копировать исходный текст дословно — только синтез
- Не добавлять информацию, отсутствующую в источниках
- wiki_* поля — только для wiki-страниц
