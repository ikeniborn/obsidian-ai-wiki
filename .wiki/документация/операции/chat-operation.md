# Chat Operation

Интерактивный чат в боковой панели для уточнения результатов Lint и Query. Запускается кнопкой в панели, а не через Command Palette.

## Назначение

Позволяет итерировать результаты операций без перезапуска агента. Доступен после Lint и Query.

## UX-поток

1. После завершения Lint или Query — в боковой панели появляется кнопка **Chat**.
2. Пользователь вводит уточняющий вопрос или инструкцию.
3. Агент обрабатывает запрос в контексте предыдущей операции.

## Ограничения платформы

Только desktop.
