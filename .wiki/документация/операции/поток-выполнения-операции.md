# Поток выполнения операции

Стандартный путь от нажатия кнопки в UI до завершения операции и обновления истории.

## Последовательность

```
User → LlmWikiView
  → WikiController.dispatch(op, args)
      ├─ isBusy()? → Notice "операция уже выполняется"
      └─ создать AbortController
         → buildAgentRunner(vaultRoot)
         → setRunning(op, args) в View
         → AgentRunner.run(RunRequest) → AsyncGenerator<RunEvent>
              loop: for await ev of runGen
                → phases/*: runIngest | runQuery | runLint | ...
                  → llm.chat.completions.create(messages, stream=true)
                  ← stream chunks
                  → yield RunEvent
                ← RunEvent
              → controller.appendEvent(ev)
              → при ev.kind == domain_created: settings.domains.push, saveSettings()
              → при ev.kind == format_preview: _pendingFormat.tempPath = ...
         → history.push(entry), saveSettings()
         → view.finish(entry)
```

## Операции по платформе

| Операция | Desktop | Mobile |
|---|---|---|
| ingest | + | - |
| query / query-save | + | + |
| lint / fix | + | - |
| init | + | - |
| chat | + | - |
| format | + | + |

## Stream-JSON протокол (claude-agent backend)

```
{ "type": "system",    "subtype": "init|error", "model": "...", "cwd": "..." }
{ "type": "assistant", "message": { "content": [{ "type": "tool_use"|"text", ... }] } }
{ "type": "user",      "message": { "content": [{ "type": "tool_result", "is_error": bool }] } }
{ "type": "result",    "duration_ms": N, "total_cost_usd": N, "result": "...", "is_error": bool }
```

Не-JSON строки (баннеры iclaude) игнорируются.

## Graph cache в query/lint

Фазы `query` и `lint` перед обращением к LLM подтягивают граф wiki через `graphCache.get(domain)` ([[wiki-graph-cache]]). Фаза `query` дополнительно вызывает `selectSeeds()` ([[wiki-seeds]]) и эмитит событие `graph_stats`.
