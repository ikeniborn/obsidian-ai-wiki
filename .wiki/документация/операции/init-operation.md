# Init Operation

Инициализирует новый wiki-домен с нуля: создаёт структуру папок и служебные файлы (`_schema.md`, `_index.md`).

## Назначение

Первый шаг при добавлении нового раздела wiki. Настраивает домен для последующего ingest корпуса источников.

## UX-поток

1. `Command Palette` → `AI Wiki: Init домена`.
2. Ввести имя домена (например, `work`).
3. Снять флаг Dry Run → запустить.
4. Плагин создаёт структуру папок и служебные файлы.
5. После init рекомендуется запустить ingest для наполнения домена.

## Ограничения платформы

Только desktop.
