# Agent Base Contract

Единый базовый системный промт (`prompts/base.md`), применяемый ко всем операциям. Решает проблему разрозненных правил поведения в фазовых промтах.

## Цель

Создать hardcoded базовый контракт, инжектируемый первым в системный промт каждой операции, до фазового промта и `## Уточнение` из пользовательских настроек.

## Порядок системного промта

```
[prompts/base.md]          ← базовый контракт (hardcoded)
[prompts/<phase>.md]       ← фазовый промт
## Уточнение               ← settings.systemPrompt (если задан)
```

## Точка инжекции

`src/phases/llm-utils.ts` → `buildChatParams()`. Импорт через esbuild text loader:

```ts
import baseContract from "../../prompts/base.md";
```

## Содержимое base.md

Три раздела:
- **Достоверность** — только на основе предоставленного контекста; не выдумывать факты
- **Формат** — возвращать ровно то, что запрошено; JSON без обёрток
- **Минимализм** — не добавлять непрошенное; не комментировать свои действия

## Out of Scope

- Изменение фазовых промтов
- Изменение `settings.systemPrompt`
- Новые настройки или UI
