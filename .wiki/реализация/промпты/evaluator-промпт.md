# Evaluator промпт (prompts/evaluator.md)

Системный промпт для оценки качества работы wiki-агента. Используется в dev mode для автоматической оценки операций через отдельный LLM-вызов.

## Основные характеристики

- **Расположение:** `prompts/evaluator.md`
- **Назначение:** Оценить результат операции по шкале 0–10

### Переменные шаблона

| Переменная | Источник |
|-----------|---------|
| `{{operation}}` | Тип операции (ingest, query, lint, init) |
| `{{task_input}}` | Входное задание операции |
| `{{result}}` | Результат операции |

### Выходной формат

```json
{"score": <0-10>, "reasoning": "<одно предложение>"}
```

### Шкала оценки

| Диапазон | Интерпретация |
|----------|--------------|
| 9–10 | Полное соответствие заданию, без ошибок |
| 7–8 | Корректен, есть незначительные недочёты |
| 5–6 | Задание выполнено частично |
| 0–4 | Не соответствует заданию или содержит ошибки |
